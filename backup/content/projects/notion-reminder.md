## 项目概述

这是一个基于Notion API开发的个人任务提醒系统，可以自动同步和提醒待办事项。

### 核心功能

1. Notion数据库同步
2. 自动任务提醒
3. 多端消息推送

### 技术栈

- Notion API
- Node.js
- 企业微信/钉钉 API
