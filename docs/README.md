# 博客文档库

这是一个个人博客文档库，按照不同的主题分类整理。

## 目录结构

```
docs/
├── posts/                         # 博客文章
│   ├── product-management/        # 产品管理相关
│   ├── tech-tools/                # 技术工具相关
│   ├── finance/                   # 财务金融相关
│   ├── insurance/                 # 保险相关
│   │   └── agriculture-insurance/ # 农业保险相关
│   ├── personal-blog/             # 个人博客建设相关
│   ├── open-source/               # 开源项目相关
│   └── family-life/               # 家庭生活相关
│       └── travel/                # 旅行计划相关
├── assets/                        # 静态资源
│   ├── images/                    # 图片资源
│   └── files/                     # 其他文件
└── drafts/                        # 草稿文章
```

## 分类说明

### 产品管理

包含产品设计、产品架构、需求分析、用户体验等相关文章。

### 技术工具

包含各种技术工具的使用教程、经验分享，如Cursor、DeepSeek、Notion、GitHub等。

### 财务金融

包含个人理财、房地产、投资等相关内容。

### 保险

包含各类保险产品研究、使用经验等内容。

#### 农业保险

包含农业保险产品设计、数字化转型等专业领域内容。

### 个人博客建设

个人博客的设计、开发、维护相关内容。

### 开源项目

开源项目的推荐、使用经验、贡献指南等内容。

### 家庭生活

家庭生活相关的内容和计划。

#### 旅行

旅行计划、游记、出行攻略等相关内容。

## 贡献指南

请遵循项目根目录下的 [CONTRIBUTING.md](/CONTRIBUTING.md) 文件中的规范进行文档贡献。
